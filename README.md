# QR Code generator

Static document QR code generator. Get in touch to try it out.

- **Blob Storage** : Firebase
- **Webapp Stack** : Python (FastApi)
- **Backend Hosting** : Vercel
- **Frontend Stack** : Python (Streamlit)
